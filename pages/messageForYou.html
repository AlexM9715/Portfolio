Message For you

Technologies Used: Python, Flask, MySQL DB

When a user first enters "Message For you", they will be greeted with a single option to "Grab A Message".

The user is then redirected to a new page where a random message will be pulled from the MySQL database. This is done by running a query to retrieve a random message. The message will populate on the screen. The user can then select a new message or create a message.

If a user decides to create a new message, they will be redirected to a form to create a message. Using Backend validations, the user can enter a message and either enter a name or remain anonymous by checking the checkbox. Once the user is finished up, they simply "Create" the message and their message is now entered into the datadase.
