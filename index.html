Projects

For When I...

Meet "For When I..."! This is a Random Song Generator based on the mood you are in. Using the Spotify API, a user will be able to search for a specific song, save the song's specific ID, and then assign it a category. Once added to the database, the user can select an option, or category, and have a random song from that category played.

Message For You

This is "Message For You". Words of encouragement and motivation are nice to hear, but someone may not always be there to give them to you. I made this site that is meant to give you a random message and hopefully with it, some motivation and encouragement to get through whatever it is you need to! Messages can be created by anyone and if you wish to stay anonymous, the option is there for you.
